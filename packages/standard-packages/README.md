zcfs:standard-packages
=========================

This packege serves as a wrapper for several other packages and has no functionality in itself:

* zcfs:standard-packages (implies some component packages)
  * zcfs:base-package
  * zcfs:file
  * zcfs:collection
  * zcfs:collection-filters
  * zcfs:access-point
  * zcfs:worker
  * zcfs:upload-http

Refer to the [CollectionFS](https://github.com/zcfs/Meteor-CollectionFS)
documentation for more information.
